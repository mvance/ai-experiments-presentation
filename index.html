Layout

Using the Available Space

Tall Card, Full Viewport

Instead of scaling slides down to a fixed aspect ratio, this deck disables Reveal’s layout engine and uses pure CSS. Each slide:

  • Fills the viewport with a flex container.
  • Contains a single tall card capped at 90vh.
  • Scrolls internally if content exceeds the visible height.

This lets you design slides that feel closer to Evernorth’s full-height content sections while still behaving like a traditional deck.

Code

Syntax Highlighting

Reveal.js Initialization

Highlight.js is wired up so code blocks are automatically styled. Here’s the initialization snippet used for this deck: 


Reveal.initialize({
  hash: true,
  width: "100%",
  height: "100%",
  margin: 0,
  minScale: 1,
  maxScale: 1,
  center: false,
  disableLayout: true,
  slideNumber: "c/t",
  transition: "fade"
});
