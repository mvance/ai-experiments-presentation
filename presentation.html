What Didn't Work

Failures and semi-failures are part of the process

Failure Type 1: Technical Complexity

Python ML Libraries for Book Highlight Tagging

  • Libraries were very tricky to get running
  • Probably needed bigger pre-tagged dataset
  • Lesson: Sometimes the infrastructure overhead isn't worth it

Failure Type 2: Wrong Tool Choice

VLC Plugin Development

  • Lots of boilerplate code required
  • Not beginner-friendly ecosystem
  • Lesson: Evaluate ecosystem friendliness early

Failure Type 3: Scope/Strategy Issues

Early Spelldle Attempts

  • Failed at "simple" backend implementations
  • Countless attempts before stability
  • Lesson: If LLM fails 2-3 times, change strategy

Embracing Failure

  • Ill-fated attempts don't seem as bothersome anymore
  • LLMs can help clean up the mess/aftermath
  • Get 2nd, 3rd opinions from different models
  • Don't be afraid to put projects on hold or abandon them

Practical Takeaways

Actionable advice you can use immediately

Getting Started Strategy

  • Start small - Don't boil the ocean
  • Keep it fun - Avoid bloat and complex hosting
  • Keep a running list - Project ideas handy
  • Circle back - Apply new lessons to old projects

When AI Helps Most

  • Logistical tasks: Debugging, dependency management
  • JSON parsing and data manipulation
  • Writing: Prompts, specs, documentation, commit messages
  • Reference implementations and PoCs
  • CLI one-liners with jq and other tools

Managing Complexity & Expectations

  • Change strategy if LLM fails 2-3 times
  • Use spec-driven development - Show examples, don't just describe
  • Get multiple opinions from different models
  • Balance: Keep it simple vs. don't reinvent the wheel

Tool Selection Wisdom

  • API calling is often easier than complex local setups
  • CDN-hosted dependencies avoid build complexity
  • Evaluate ecosystem friendliness early
  • Don't be afraid to try new languages, tools, APIs

What's Next?

  • Spec-driven development tools: Spec-Kit, Kiro, Tessl, BMad Method

  • Test-driven development with AI assistance
